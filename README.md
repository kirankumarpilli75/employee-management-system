# Employee Management System

A web-based Employee Management System developed using Python Django.  
This application helps manage employee records with user authentication and CRUD operations.

## Features

- User Registration and Login
- User Authentication
- Add Employee Details
- Update Employee Information
- Delete Employee Records
- View Employee Details
- Profile Photo Upload
- Responsive User Interface

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite
